# Vrinda Store Sales Analysis (Excel Project)

## Project Overview
This project analyzes sales data of Vrinda Store for the year 2022 using Microsoft Excel.

The goal is to identify sales trends, customer behavior, and key business insights using Excel dashboards and pivot tables.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Creation

## Analysis Performed
- Monthly Orders vs Sales Analysis
- Gender-based Customer Analysis
- Order Status Distribution
- Top States by Revenue
- Sales Channel Performance

## Key Insights
- Women customers contribute the majority of orders.
- Amazon and Myntra are the top sales channels.
- Certain states generate the highest revenue.
- Most orders are successfully delivered.
